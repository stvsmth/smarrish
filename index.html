Vimium Shortcuts

Navigating the page

  1. f Open a link in the current tab
  2. F Open a link in a new tab
  3. <a-f> Open multiple links in a new tab
  4. j Scroll down
  5. k Scroll up
  6. h Scroll left
  7. l Scroll right
  8. gg Scroll to the top of the page
  9. G Scroll to the bottom of the page
  10. zH Scroll all the way to the left
  11. zL Scroll all the way to the right
  12. d Scroll a page down
  13. u Scroll a page up
  14. r Reload current frame
  15. <a-r> Reload N tabs
  16. R Reload N-th tab
  17. gs View page source
  18. yy Copy page's info
  19. <f2> Blur activeElement or refocus it
  20. gi Focus the N-th visible text box on the page and cycle using tab
  21. yf Copy a link URL to the clipboard
  22. p Open the clipboard's URL in the current tab
  23. P Open the clipboard's URL in N new tabs
  24. gu Go up the URL hierarchy
  25. gU Go to root of current URL hierarchy
  26. yi Copy image to the clipboard
  27. yv Select text and enter visual mode
  28. <f1> Simulate backspace once if focused
  29. ]] Follow the link labeled next or ">"
  30. [[ Follow the link labeled previous or "<"
  31. gf Cycle forward to the next frame on the page
  32. gF Select the tab's main/top frame
  33. i Enter insert mode
  34. v Enter visual mode
  35. V Enter visual line mode
  36. m Create a new mark
  37. ` Go to a mark

Using the Vomnibar

  1. o Open URL, bookmark, or history entry
  2. O Open URL, history, â€¦ in a new tab
  3. b Open a bookmark
  4. B Open a bookmark in a new tab
  5. T Search through your open tabs
  6. ge Edit the current URL
  7. gE Edit the current URL and open in a new tab
  8. gn Toggle styles of vomnibar page

Using find

  1. / Enter find mode
  2. n Cycle forward to the next find match
  3. N Cycle backward to the previous find match
  4. <a-n> Find the second or even earlier query words

Navigating history

  1. H Go back in history
  2. L Go forward in history
  3. <a-s-r> Reopen current page

Manipulating tabs

  1. K gt Go one tab right
  2. J gT Go one tab left
  3. g0 Go to the first N-th tab
  4. g$ Go to the last N-th tab
  5. t <a-t> Create new tabs
  6. yt Duplicate current tab for N times
  7. x Close N tabs
  8. X Restore closed tabs
  9. W Move tab to next window
  10. <c -\> Move N tabs to new window
  11. <a-p> Pin or unpin N tabs
  12. <a-m> Mute or unmute current tab
  13. ^ Go to previously-visited tab on current window
  14. << Move tab to the left
  15. >> Move tab to the right
